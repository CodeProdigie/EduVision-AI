# PaperVideo

A scalable, production-ready architecture foundation for an AI-powered educational platform.

## Structure

- Frontend: React + Vite + Tailwind + TanStack Query + Zustand
- Backend: Node + Express + MongoDB + Mongoose + JWT + Multer-ready foundation
- Routing: protected routes, layouts, responsive sidebar, and dark mode

## Getting started

### Frontend

- Install dependencies with Bun: `bun install`
- Start the app: `bun run dev`

### Backend

- Change into the server directory: `cd server`
- Install server dependencies: `bun install`
- Start the API: `bun run dev`

## Environment

Copy `.env.example` to `.env` and adjust the values for local development.
