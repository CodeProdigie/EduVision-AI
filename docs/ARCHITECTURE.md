# PaperVideo Architecture

## Overview

PaperVideo is a modular educational platform designed for paper upload, AI-assisted processing, video generation, and learning support.

## Frontend

- React + Vite
- TailwindCSS
- TanStack Query
- Zustand
- Reusable components and hooks

## Backend

- Node.js + Express
- MongoDB + Mongoose
- JWT auth
- Multer-based uploads
- Modular AI and video services

## Production concerns

- Environment-based configuration
- Logging and error handling
- Rate limiting
- Reusable loading and empty states
- Responsive and accessible UI patterns
- Docker-ready setup
